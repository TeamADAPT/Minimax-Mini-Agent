## Objective

Extract adaptable chat/session, analytics/logs, and collaboration workflow ideas from local Hermes UI candidates and open-source design tools without straight copying.

## Sources

- `nesquena/hermes-webui`
- `EKKOLearnAI/hermes-web-ui`
- Penpot
- Excalidraw
- CryptPad
- OpenPencil/Pencil
- Skills: `design-md`, `architecture-diagram`, `popular-web-designs`, `dogfood`, `sketch`

## Steps

1. Move this folder to `ops/in_progress/`.
2. Verify candidate apps are runnable or document current launch command.
3. Inspect chat/session UX from `nesquena/hermes-webui`.
4. Inspect analytics/logs UX from `EKKOLearnAI/hermes-web-ui`.
5. Inspect design-collaboration affordances from installed tools.
6. Create a pattern inventory with screenshots where useful.
7. Convert patterns into implementation task folders under `ops/to_do/`.
8. Write `completion_report.md`.
9. Move this folder to `ops/completed/`.
10. Commit and push.

## Acceptance

- Pattern inventory exists.
- New implementation task folders exist.
- No straight-copy implementation is proposed.
